# Dotfiles for MAC

## Requirements

Ensure, that the following items are installed:

- Homebrew
- Neovim
- Nerdfont
- Kitty
- Zsh
- Tmux
- SDKMAN
- Miniconda
- fzf, rg

## Terminal

- Emulator: Wezterm
- Shell: zsh
