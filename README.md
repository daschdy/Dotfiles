## Requirements

Ensure, that the following items are installed:

- Homebrew
- Neovim
- Nerdfont
- Kitty
- Zsh, especially Zap
- Tmux
- SDKMAN
- Miniconda
- NodeJS
- fzf, rg

## Terminal

- Emulator: Kitty
- Shell: zsh
- Plugins: zap
